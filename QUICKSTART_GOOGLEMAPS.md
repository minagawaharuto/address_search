# Google Maps API版 - クイックスタートガイド

最速で住所検索システムを使い始めるためのガイドです。

## 🚀 3ステップで開始

### ステップ1: Google Maps APIキーを取得（5分）

1. https://console.cloud.google.com/ にアクセス
2. プロジェクトを作成
3. **Places API** を有効化
4. APIキーを作成してコピー

詳細は [GOOGLE_MAPS_API_SETUP.md](GOOGLE_MAPS_API_SETUP.md) を参照

### ステップ2: APIキーを設定（1分）

`config.txt` ファイルを作成：

```
GOOGLE_MAPS_API_KEY=あなたのAPIキーをここに貼り付け
```

### ステップ3: アプリを起動（1分）

```cmd
# 依存パッケージをインストール
pip install -r requirements_googlemaps.txt

# アプリを起動
python app_googlemaps.py
```

ブラウザで http://localhost:5001 を開く

## ❗ よくある質問

### Q1: APIキーの取得にクレジットカードが必要ですか？

**A**: はい、必要です。ただし、**無料枠内なら課金されません**。

### Q2: 無料枠を超えるとどうなりますか？

**A**: 予算アラートを設定していれば、メールで通知されます。自動で課金されるため、予算設定を推奨します。

### Q3: Selenium版と併用できますか？

**A**: はい、できます。両方のアプリが異なるポート（5000と5001）で動作します。

### Q4: どちらを使うべきですか？

**A**: **Google Maps API版を推奨**します。精度が高く、安定しています。

### Q5: APIキーが漏洩したらどうすればいいですか？

**A**: すぐにGoogle Cloud Consoleでキーを削除し、新しいキーを作成してください。

## 🆘 トラブルシューティング

### エラー: API Key 未設定

```cmd
# config.txt の内容を確認
type config.txt

# 正しい形式:
# GOOGLE_MAPS_API_KEY=AIzaSyDXXXXXXXXXXXXXXXXX
```

### エラー: REQUEST_DENIED

Places APIを有効化してください：
1. https://console.cloud.google.com/
2. 「APIとサービス」→「ライブラリ」
3. 「Places API」を検索して有効化

### 住所が見つからない

1. 会社名のスペルを確認
2. Google Mapsで手動検索して、店舗が登録されているか確認
3. 支店名の表記を変更（例: 「店」→「支店」）
