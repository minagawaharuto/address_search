# 住所検索システム

会社名と支店名から自動的に住所を検索し、Excelファイルに追加するWebアプリケーションです。

## 機能

- 会社名を入力
- 支店名が記載されたExcelファイルをアップロード
- Google検索を使って自動的に住所を検索
- 同じExcelファイルに「住所」列を追加してダウンロード

## 必要な環境

- Python 3.8以上
- Google Chrome（Seleniumで使用）

## セットアップ手順

### 1. Pythonのインストール確認

```bash
python --version
```

Python 3.8以上がインストールされていることを確認してください。

### 2. 依存パッケージのインストール

プロジェクトフォルダで以下のコマンドを実行：

```bash
pip install -r requirements.txt
```

### 3. Google Chromeのインストール

Seleniumが使用するため、Google Chromeがインストールされている必要があります。
ChromeDriverは自動的にダウンロードされます（webdriver-manager使用）。

## 使い方

### 1. アプリケーションの起動

```bash
python app.py
```

起動すると、以下のメッセージが表示されます：

```
 * Running on http://0.0.0.0:5000
```

### 2. ブラウザでアクセス

ブラウザで以下のURLにアクセス：

```
http://localhost:5000
```

### 3. ファイルのアップロードと検索

1. **会社名を入力**
   - 例: 株式会社サンプル

2. **Excelファイルを選択**
   - 支店名が記載された.xlsxファイルを選択
   - Excelファイルの形式例：

   | 支店名 |
   |--------|
   | 東京支店 |
   | 大阪支店 |
   | 名古屋支店 |

3. **検索開始ボタンをクリック**
   - 自動的に各支店の住所を検索
   - 処理には数分かかる場合があります

4. **結果のダウンロード**
   - 処理完了後、自動的にファイルがダウンロードされます
   - ファイル名: `住所追加_元のファイル名.xlsx`

## Excelファイルの形式

### 入力ファイル

最低限、支店名が記載された列が必要です：

| 支店名 |
|--------|
| 東京支店 |
| 大阪支店 |

列名は「支店名」「支店」などに対応していますが、列名がなくても最初の列を支店名として使用します。

### 出力ファイル

住所列が追加されます：

| 支店名 | 住所 |
|--------|------|
| 東京支店 | 東京都千代田区... |
| 大阪支店 | 大阪府大阪市... |

## 注意事項

### 検索精度について

- Google検索結果から住所を抽出するため、100%の精度は保証されません
- 検索結果に住所が含まれていない場合、「住所が見つかりませんでした」と表示されます
- 検索クエリは「会社名 + 支店名 + 住所」の形式で行われます

### 利用制限について

- Google検索のスクレイピングはGoogleの利用規約で推奨されていません
- 大量の検索を短時間で行うとIPアドレスがブロックされる可能性があります
- 検索間隔を3秒空けるように設定していますが、使用は自己責任でお願いします

### 推奨事項

- **商用利用や大規模な利用の場合**
  - Google Maps Geocoding APIの使用を推奨します
  - 月28,000リクエストまで無料で利用可能
  - より高精度で安定した結果が得られます

## トラブルシューティング

### ChromeDriverのエラー

```
ChromeDriver not found
```

→ Google Chromeがインストールされているか確認してください

### ファイルアップロードエラー

```
ファイルが選択されていません
```

→ .xlsx形式のExcelファイルを選択してください

### 住所が見つからない

- 会社名のスペルを確認
- 支店名が正確か確認
- 一般的でない表記の場合は手動で確認が必要な場合があります

- **Flask**: Webフレームワーク

### カスタマイズ

検索待機時間を変更する場合は、`app.py`の以下の行を編集：

```python
time.sleep(3)  # 秒数を変更
```

検索クエリを変更する場合は、`search_address`関数内の以下の行を編集：

```python
search_query = f"{company_name} {branch_name} 住所"
```

## サポート

1. Pythonのバージョン（3.8以上）
2. 必要なパッケージがインストールされているか
3. Google Chromeがインストールされているか
4. Excelファイルの形式が正しいか

それでも解決しない場合は、エラーメッセージを確認してください。
